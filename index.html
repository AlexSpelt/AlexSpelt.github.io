Ervaring waarom ik denk dat HBO-ICT de studie voor mij is

Ik ben al lang gemotiveerd rond het onderwerp ICT. Vanaf toen ik 13 was ben ik begonnen met software schrijven en computers bouwen.

In eerstenstantie heb ik in Lua programma's gemaakt. Ik begon langzaam met Java mods te schrijven voor Minecraft. Tot die tijd was dat gewoon nog een hobby. Die ervaring nam ik mee op een stage. Ik moest op de middelbare school 2 weken stage lopen. Die stage heb ik gelopen bij Sflex Sign Software gelopen. Na die stage ben ik daar aangenomen als Front-End ontwikkelaar. ik werk ondertussen 3 jaar voor sflex sign software en ben nu Full Stack web developer.

Voor deze opleiding heb ik de opleiding Media- applicatie ontwikkelaar MBO Niveau 4 gevold. Aan dat diploma heb ik niet zo veel, maar het is wel een goede overstap naar deze opleiding. Met een HBO ICT diploma heb ik ook gegarandeerd een baan, daarintegen kan ik niet hetzelfde zeggen voor mijn MBO diploma.
